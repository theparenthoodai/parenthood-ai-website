# Guide "example build" proof images

Drop one PNG per guide here, named `<slug>-example.png`. The guide page shows it
in the "The example build" section; until the file exists, the page shows a
tasteful "example image coming soon" placeholder.

Expected filenames:
- youve-tried-ai-example.png
- takeout-planner-example.png
- scholarship-finder-example.png
- summer-weekends-example.png
- daily-brief-kit-example.png
- the-comeback-plan-example.png

Suggested width ~1200px, landscape. After adding images, run `python build.py`
and commit.
